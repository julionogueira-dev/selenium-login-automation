# 🚀 Selenium Login Automation

Projeto de automação web desenvolvido em Python e Selenium para automatizar o processo de autenticação utilizando o navegador Opera GX.

---

## 📚 Objetivo

Este projeto foi desenvolvido com fins didáticos para praticar:
- Automação web e testes funcionais.
- Manipulação avançada de elementos HTML com Selenium WebDriver.
- Gerenciamento de esperas explícitas e implícitas no Python.

---

## ✨ Funcionalidades

- **Inicialização Customizada:** Abertura automática mapeada para o navegador Opera GX.
- **Login Automatizado:** Preenchimento inteligente de credenciais e interação com botões de envio.
- **Agendamento Prévio:** Execução e espera programada baseada em horários específicos.
- **Robustez:** Interação segura com elementos da página (evitando quebras por lentidão).

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python
- **Framework Principal:** Selenium WebDriver
- **Navegador Alvo:** Opera GX (via OperaDriver)
